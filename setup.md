# Setup Procedure

## Windows

- Get NVM for Windows: https://github.com/coreybutler/nvm-windows/releases
- `nvm install 22.4.1`
- `nvm use 22.4.1`
- `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned`
- `node --version`
- `npm --version`
- Setup node/npm in IDEA
- `npm install`
- `cp env.example .env`
- Need to seed DB
- `npm setup`
- `npm dev`
